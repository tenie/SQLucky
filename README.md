# SQLucky

> 简介:
    数据库工具
---

> 下载:
   [链接](https://github.com/tenie/SQLucky/releases/)
---

> 依赖:

    Java 20 
    maven  latest
    git    latest
    
> 快速启动:
   
     
    $ git clone git@github.com:tenie/SQLucky.git
    $ cd  SQLucky
    $ mvn -N install
    $ mvn  install
    $ cd app
    $ mvn javafx:run
