# SQLucky

> 简介:
    数据库工具
---

> 下载:
   [链接](https://github.com/tenie/SQLucky/releases/)
---

> 依赖:

    Java 17 
    maven  latest
    git    latest
    
> 快速启动:
   
     
    $ git clone git@github.com:tenie/SQLucky.git
    $ cd  SQLucky
    $ mvn -N install
    $ mvn  install
    $ cd app
    $ mvn javafx:run

> change log:

    2.0.0 更新:
        1. 实现插件功能
        2. 插件方式加载各种类数据库操作API实现
        3. 添加脚本栏, 更便捷管理脚本
        4. 添加笔记栏, 打开指定的笔记目录后, 直接编辑笔记文本.
        5. 直接运行光标所在行sql
        6. 编辑器中右键菜单添加编辑功能
